# Astrologic - Backend

Proyecto backend desarrollado con Spring Boot para la gestión de eventos meteorológicos en la plataforma Astrologic.

## Descripción

Astrologic es un sistema que permite registrar y organizar eventos climáticos como tormentas, lluvias y otros fenómenos meteorológicos. El proyecto está enfocado en la estructura backend utilizando buenas prácticas básicas de desarrollo.

## Tecnologías utilizadas

- Java
- Spring Boot
- Spring Data JPA
- H2 Database

## Estructura del proyecto

- Modelo: representación de los datos del evento climático
- Repositorio: acceso a la base de datos
- DTO: transferencia de datos sin exponer el modelo
- Mapper: conversión entre modelo y DTO
- Prueba: verificación del funcionamiento del sistema

## Funcionalidades

- Creación de eventos climáticos
- Conversión de datos entre capas (modelo y DTO)
- Persistencia de datos mediante JPA
- Pruebas básicas de funcionamiento

## Uso de herramientas de apoyo

Durante el desarrollo del proyecto se realizaron correcciones con apoyo de herramientas de inteligencia artificial. Estas se utilizaron principalmente para:

- Organización del DTO
- Estructura del mapper
- Mejora en la separación de capas
- Documentación del proyecto
